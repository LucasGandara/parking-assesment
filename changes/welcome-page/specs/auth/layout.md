# Auth Layout

## Overview

Defines the shared two-panel visual shell rendered on both the login and
register pages.

## Access

> Public

## Requirements

### Structure

- The system SHALL render a two-panel layout on the `/login` and `/register`
  pages.
- The system SHALL render a left panel containing the ParkSmart AI logo, the
  tagline "Intelligent Parking Management System", a list of feature
  highlights, and a parking-themed image.
- The system SHALL render a right panel containing the authentication form
  for the current page.

### Footer

- The system SHALL render a footer on both the `/login` and `/register`
  pages with the text "Need help? Contact your building administrator."
- The system SHALL render the footer contact text as a hyperlink pointing
  to a configurable administrator email address.
